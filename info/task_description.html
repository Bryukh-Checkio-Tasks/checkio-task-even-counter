You are given a number (integer). You should count even digits in this number. Zero (0) is even.

For example: 123456 contains 3 even digits -- 2, 4 ad 6.

Input: A number (int).

Output: The quantity of even digits (int).

Example: 

checkio(123456) == 3
assert checkio(111111) == 0
assert checkio(02468) == 5
